# 🩺 Medical Risk Prediction System

An end-to-end Machine Learning project that predicts the risk of heart disease using patient health data.  
Built with a focus on **accuracy, interpretability, and real-world usability**.

---

## 🚀 Live Demo
👉 https://your-app-name.streamlit.app

---

## 📌 Features

- 🔍 Predicts heart disease risk using patient health metrics  
- 📊 Provides probability score for better decision-making  
- 🧠 Identifies key risk factors using feature importance  
- 🌐 Interactive web app built with Streamlit  
- ⚡ Fast and lightweight deployment  

---

## 🛠️ Tech Stack

- Python  
- Pandas, NumPy  
- Scikit-learn (Random Forest)  
- Matplotlib  
- Streamlit  

---

## 🧠 Machine Learning Workflow

1. Data Collection (Heart Disease Dataset)
2. Data Preprocessing
3. Feature Scaling
4. Train-Test Split
5. Model Training (Random Forest)
6. Hyperparameter Tuning (GridSearchCV)
7. Model Evaluation (Accuracy, ROC-AUC)
8. Model Deployment using Streamlit

---

## 📊 Model Performance

- **Accuracy:** ~85%  
- **ROC-AUC Score:** ~0.93  
- **Balanced Precision-Recall Performance**

---

## 🔍 Key Insights (Feature Importance)

The model identifies the following as most important features:

- **ca** – Number of major vessels (blockage indicator)  
- **thal** – Blood disorder / perfusion defect  
- **oldpeak** – ST depression during exercise  

These align closely with real-world clinical indicators.
